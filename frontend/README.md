# Perfume Store Management System - Frontend

واجهة نظام إدارة متجر العطور مبنية باستخدام Vue.js 3 و TailwindCSS

## المتطلبات

- Node.js >= 18
- npm أو yarn

## التثبيت

```bash
npm install
```

## التشغيل

```bash
npm run dev
```

سيعمل التطبيق على: http://localhost:5173

## البناء للإنتاج

```bash
npm run build
```
